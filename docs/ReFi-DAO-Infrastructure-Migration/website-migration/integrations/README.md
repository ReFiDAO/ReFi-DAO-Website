# Integration Documentation

This directory contains guides for integrating external services with the Quartz website.

## Available Integrations

### RSS Feed Integration
**File:** `rss-feed-integration.md`

Guide for integrating Ghost blog RSS feed to display latest posts on the Media page.

**Status:** Ready for implementation after Ghost migration

### Newsletter Integration
**File:** `newsletter-integration.md`

Guide for integrating newsletter service (Tally.so, Mailchimp, etc.) for email subscriptions.

**Status:** Ready for implementation

## Integration Principles

1. **Keep it Simple:** Prefer static content and simple embeds
2. **External Services:** Use external services for dynamic content
3. **Performance:** Minimize JavaScript and external dependencies
4. **Accessibility:** Ensure all integrations are accessible
